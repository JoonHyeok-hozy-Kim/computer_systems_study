# Types of Linux

## Concept) Server Linux
- Props.)
  - No GUI
  - Less SW installed
  - Less to keep updated
  - Designed for throughput and reliability
  - Multiple redundancies
- Linux Usages)
  - Databases
  - Web Servers
  - Email Servers

<br>

## Concept) Desktop Linux
- Props.)
  - GUI
  - Low cost (Free distributions)
  - No need to upgrade
- Distributions)
  - Fedora
  - Ubuntu
  - CentOS

<br>

## Concept) Mobile Linux
- Props.)
  - Designed for mobile devices
  - Touch function supported
- Distributions
  - Android (open source)
  - iOS
  - Ubuntu Touch
  - postmarketOS
  - Sailfish OS
  - Mobian
