# Running Containers

### List Images
* Old
  ```
  docker images
  ```
* New
  ```
  docker image ls
  ```

<br>

### Run (Instantiate) images
* Old
  ```
  docker run <image_name>
  ```
* New
  ```
  docker container run <image_name>
  ```

#### cf.) Who's naming containers?
* Moby Project : https://mobyproject.org/
